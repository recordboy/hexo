반올림  
`Math.round();`
```javascript
var num = 3.5;
console.log(Math.round(num)); // 4
```

올림  
`Math.ceil();`
```javascript
var num = 3.5;
console.log(Math.ceil(num)); // 4
```

내림  
`Math.floor();`
```javascript
var num = 3.5;
console.log(Math.floor(num)); // 3
```
